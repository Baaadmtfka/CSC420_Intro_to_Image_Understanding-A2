Dataset:
	SDDsubset and DBIsubset are in a folder named Dataset, which is compressed into a single zip file: Dataset.zip.
	The zip file is uploaded to Google Drive: https://drive.google.com/drive/folders/1CcxA2PjtrHGwSHvd5kLpVFeBF-EEHge1?usp=sharing

a2-implementation.ipynb:
	The implementation code for Part II.
	Dataset was loaded by mounting to my Google Drive, but can switch to local directory for each section requiring access to dataset. 
	Nota that the zip file after unzip in local directory will be of structure: /Dataset/Dataset/SDDsubset or DBIsubset, a duplicated root directory.
	The entirety of the notebook was run on Google Colab, with T4 High-RAM GPU. It used up to 14.7 GB of GPU RAM.
